<h1 style="margin:0;">Commands</h1>
<p>/getshopspawner - Gives you item shop spawner</p>

<hr/>

<p>Cannons - Buy cannon from item shop. Stand on top a gold block and drop the cannon.</p>
<p>Flintknock - Works like a normal Bow</p>
